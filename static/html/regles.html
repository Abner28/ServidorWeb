Les Règles

1) Le joueur doit lancer les cinq dés une première fois ;

2) Le score du joueur aura la somme de chaque dé ;

3) Si le joueur n'êtes pas satisfait de la somme, il / elle pourra relancer les dés cinq fois ;

4) Une fois le joueur choisit pour "enregistrer le score", il / elle ne peut pas
jouer plus et le score aura la somme des derniers dés lancée.
